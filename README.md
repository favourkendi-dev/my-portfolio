### Project Title
My Portfolio
### Installation
1. Clone the Repository :
``` bash
git clone  git@github.com:favourkendi-dev/my-portfolio.git

```

2. Install Independencies
``` bash
npm install

```

### Usage
- To run the project ,use the following commands
``` bash
npm start

```

#### Features

1. Addition
2. Subtraction
3. Division
4. Multiplication
5. History Retrival
